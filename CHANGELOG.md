# Changelog

## [v1.2.1] — 2026-04-15

### 버그 수정
- 페이지 초기 로드 시 로그인 전 Firestore 접근으로 `Missing or insufficient permissions` 오류 발생 → 시나리오 로드를 인증 성공 후로 이동
- `signInWithGoogle()`에서 제거된 `auth-error` 요소 참조로 `null textContent` 오류 발생 → `showDeniedCard()`로 대체

## [v1.2.0] — 2026-04-15

### 보안
- 구글 로그인 기반 접근 제어 추가 (Firebase Authentication)
- 허용 도메인 제한: @liveklass.com, @futureschole.com 계정만 접속 가능
- 차단 목록(blocked_users) 연동: Firestore에 등록된 계정 즉시 차단
- 미인증 상태에서 앱 전체 차단 (로그인 오버레이)
- 접근 불가 계정 로그인 시 전용 오류 화면 표시 + 다른 계정으로 시도 버튼
- Firestore Security Rules 강화: 인증된 사용자만 읽기/쓰기, 삭제는 관리자만

### 권한
- 관리자 계정 구분 추가 (admins Firestore 컬렉션 기반)
- 관리자만 삭제 버튼 및 카드 체크박스 표시
- 비관리자는 저장·조회만 가능

### 작성자 연동
- 저장 모달 작성자 드롭다운 제거
- 작성자 자동 지정: 로그인한 구글 계정 표시 이름

### 데이터 보호
- 소프트 삭제 적용: 삭제 시 문서 제거 대신 deleted 플래그 처리
- 삭제 기록 보존: deletedBy(삭제자 이메일), deletedAt(삭제 시각) 자동 기록
- 헤더에 로그인 계정 표시 이름 + 로그아웃 버튼 추가

### UX
- 시나리오 탭 로딩 중 스피너 + 안내 문구 표시

## [v1.1.0] — 2026-04-09

### 추가
- 시나리오 카드에 작성자 배지 표시
- 작성자 필터 (이름별 필터링)
- 작성자 정보 localStorage 기억 (재입력 불필요)
- 시나리오 카드 체크박스 선택 삭제 (선택 삭제 / 전체 삭제 구분)

### 개선
- 시나리오 카드 레이아웃 개선: 배지 → 런칭명 순서로 가독성 향상
- 선택 삭제(주황) / 전체 삭제(빨강) 버튼 색상 구분으로 역할 명확화
- 비교 분석 탭을 시나리오 비교 탭에 통합 (3탭 → 통합 구조)
- 시나리오 카드에 시뮬 대비 실적 증감 표시
- 실적 탭 KPI 지표 개선 및 %p(퍼센트포인트) 단위 수정
- primary 컬러 오렌지 → 인디고로 변경 (음수 빨간색과 시각 분리)

### 인프라
- localStorage → Firebase Firestore 전환 (멀티유저 지원)
- 작성자 드롭다운으로 입력 관리
