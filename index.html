사용 방법
시뮬레이션 탭과 실적 입력 탭에 데이터를 입력하면 자동으로 비교됩니다. 저장 없이 현재 세션의 입력 값을 즉시 비교합니다.

⚠️

주의사항
이 앱은 각 브라우저에 독립적으로 저장됩니다. 같은 기기에서 탭을 여러 개 열면 시나리오 데이터가 섞일 수 있습니다. 한 기기 — 한 탭으로 사용하세요.

📊 시뮬레이션 vs 실적 비교
📐

사용 방법
저장한 시뮬레이션과 실적을 한눈에 비교합니다. 2개 이상 저장되면 하단에 비교표가 자동으로 표시됩니다. 시뮬레이션 카드의 불러오기로 해당 시나리오를 시뮬레이션 탭에 다시 로드할 수 있습니다. 실적 카드는 불러오기를 지원하지 않습니다.
